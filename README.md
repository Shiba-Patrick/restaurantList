# 餐廳清單

此專案提供使用者觀看全部餐廳清單，使用者可以搜尋關鍵字查詢餐廳並且查看餐廳詳細資訊

## 功能說明

* 使用者可以點選餐廳查看其詳細資訊，例如：地址、電話、地圖等等。
* 使用者可以使用關鍵字搜尋想要的餐廳（店名以及類別）
